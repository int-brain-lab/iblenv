brainbox.behavior.pyschofitÔÉÅ

(DEPRECATED) The psychofit toolbox contains tools to fit two-alternative psychometric data. The fitting is done using maximal likelihood estimation: one assumes that the responses of the subject are given by a binomial distribution whose mean is given by the psychometric function. The data can be expressed in fraction correct (from .5 to 1) or in fraction of one specific choice (from 0 to 1). To fit them you can use these functions:

  • weibull50: Weibull function from 0.5 to 1, with lapse rate

  • weibull: Weibull function from 0 to 1, with lapse rate

  • erf_psycho: erf function from 0 to 1, with lapse rate

  • erf_psycho_2gammas: erf function from 0 to 1, with two lapse rates

Functions in the toolbox are:

  • mle_fit_psycho: Maximumum likelihood fit of psychometric function

  • neg_likelihood: Negative likelihood of a psychometric function

For more info, see:

  • Examples: Examples of use of psychofit toolbox

Matteo Carandini, 2000-2015

NB: USE THE PSYCHOFIT PIP PACKAGE INSTEAD.

Negative likelihood of a psychometric function.

Parameters:

  • pars ‚Äì Model parameters [threshold, slope, gamma], or if using the ‚Äòerf_psycho_2gammas‚Äô model append a second gamma value.

  • data ‚Äì 3 x n matrix where first row corresponds to stim levels, the second to number of trials for each stim level (int), the third to proportion correct / proportion rightward (float between 0 and 1)

  • P_model ‚Äì The psychometric function. Possibilities include ‚Äòweibull‚Äô (DEFAULT), ‚Äòweibull50‚Äô, ‚Äòerf_psycho‚Äô and ‚Äòerf_psycho_2gammas‚Äô

  • parmin ‚Äì Minimum bound for parameters. If None, some reasonable defaults are used

  • parmax ‚Äì Maximum bound for parameters. If None, some reasonable defaults are used

Returns:

The likelihood of the parameters. The equation is:

  • sum(nn.*(pp.*log10(P_model)+(1-pp).*log10(1-P_model)))

See the the appendix of Watson, A.B. (1979). Probability summation over time. Vision Res 19, 515-522.
