setup_logger(name='ibl', level=0, file=None, no_color=False)[source]ÔÉÅ

Set up a log for IBL packages.

Uses date time, calling function and distinct colours for levels. Sets the name if not set already and add a stream handler. If the stream handler already exists, does not duplicate. The naming/level allows not to interfere with third-party libraries when setting level.

Parameters:

  • name (str) ‚Äì Log name, should be set to the root package name for consistent logging throughout the app.

  • level (str, int) ‚Äì The logging level (defaults to NOTSET, which inherits the parent log level)

  • file (bool, str, pathlib.Path) ‚Äì If True, a file handler is added with the default file location, otherwise a log file path may be passed.

  • no_color (bool) ‚Äì If true the colour log is deactivated. May be useful when directing the std out to a file.

Returns:

The configured log.

Return type:

logging.Logger, logging.RootLogger
