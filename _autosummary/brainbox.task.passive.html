Returns

  • rf_map_times (time of each receptive field map frame np.array(len(stim_frames))

  • rf_map_pos (unique position of each pixel on scree np.array(len(x_pos), len(y_pos)))

  • rf_stim_frames (for each pixel on screen stores array of stimulus frames where stim onset)

  • occured. For both white squares ‘on’ and black squares ‘off’

Compute receptive field map for each stimulus onset binned across depth :param rf_map_times: :param rf_map_pos: :param rf_stim_frames: :param spike_times: :type spike_times: array of spike times :param spike_depths: :type spike_depths: array of spike depths along probe :param t_bin: :type t_bin: bin size along time dimension :param d_bin: :type d_bin: bin size along depth dimension :param pre_stim: :type pre_stim: time period before rf map stim onset to epoch around :param post_stim: :type post_stim: time period after rf map onset to epoch around :param y_lim: :type y_lim: values to limit to in depth direction :param x_lim: :type x_lim: values to limit in time direction

Returns

  • rfmap (receptive field map for ‘on’ ‘off’ stimuli.)

  • Each rfmap has shape (depths, x_pos, y_pos, epoch_window)

  • depths (depths between which receptive field map has been computed)
