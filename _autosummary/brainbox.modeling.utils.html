class SequentialSelector(model, n_features_to_select=None, direction='forward', scoring=None)[source]¶

Bases: object

__init__(model, n_features_to_select=None, direction='forward', scoring=None)[source]¶

Sequential feature selection for neural models

Parameters

  • model (brainbox.modeling.neural_model.NeuralModel) – Any class which inherits NeuralModel and has already been instantiated.

  • n_features_to_select (int, optional) – Number of covariates to select. When None, will sequentially fit all parameters and store the associated scores. By default None

  • direction (str, optional) – Direction of sequential selection. ‘forward’ indicates model will be built from 1 regressor up, while ‘backward’ indicates regrssors will be removed one at a time until n_features_to_select is reached or 1 regressor remains. By default ‘forward’

  • scoring (str, optional) – Scoring function to use. Must be a valid argument to the subclass of NeuralModel passed to SequentialSelector. By default None
