rolling_window(x, window_len=11, window='blackman')[source]ÔÉÅ

Smooth the data using a window with requested size.

This method is based on the convolution of a scaled window with the signal. The signal is prepared by introducing reflected copies of the signal (with the window size) in both ends so that transient parts are minimized in the beginning and end part of the output signal.

Parameters

  • x (list or numpy.array) ‚Äì The input signal

  • window_len (int, optional) ‚Äì The dimension of the smoothing window, should be an odd integer, defaults to 11

  • window (str, optional) ‚Äì The type of window from [‚Äòflat‚Äô, ‚Äòhanning‚Äô, ‚Äòhamming‚Äô, ‚Äòbartlett‚Äô, ‚Äòblackman‚Äô] flat window will produce a moving average smoothing, defaults to ‚Äòblackman‚Äô

  • window (int (odd)) ‚Äì Window length of datapoints. Must be odd and smaller than x

  • polynom (int) ‚Äì The order of polynom used. Must be smaller than the window size
