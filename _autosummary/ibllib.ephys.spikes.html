detection(data, fs, h, detect_threshold=-4, time_tol=0.002, distance_threshold_um=70)[source]ÔÉÅ

Detects and de-duplicates negative voltage spikes based on voltage thresholding. The de-duplication step locks in maximum amplitude events. To account for collisions the amplitude is assumed to be decaying from the peak. If this is a multipeak event, each is labeled as a spike.

Parameters:

  • data ‚Äì 2D numpy array nsamples x nchannels

  • fs ‚Äì sampling frequency (Hz)

  • h ‚Äì dictionary with neuropixel geometry header: see. neuropixel.trace_header

  • detect_threshold ‚Äì negative value below which the voltage is considered to be a spike

  • time_tol ‚Äì time in seconds for which samples before and after are assumed to be part of the spike

  • distance_threshold_um ‚Äì distance for which exceeding threshold values are assumed to part of the same spike
